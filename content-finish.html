Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input Content - Finish Line into the pop-up and select OK

Content - Conflict Resolution

FOMOD Install Instructions:

  • Choose all files that apply to what mods you have installed from the guide

- Resolves conflicts between the mods from this module and the previous modules

Content - Load Order.txt

Special install instructions:

  • Extract loadorder.txt to Mod Organizer 2/profiles/Viva New Vegas - Content
  • Once the file is extracted, right-click in the left pane of MO2 and select All Mods -> Refresh

- Automatically ensures your load order is correct for the Core module
NOTE: This load order assumes you followed the entire guide with all the recommended options. For example, if you installed your own mods, those plugin-ins will be moved to the bottom of the LO and need to be manually moved back into the correct spot after the new load order is applied. It will still work perfectly even if you left out some mods too

Using zMerge

  • Verify Content - Conflict Resolution.esp is in the correct spot in your load order
  • Run zEdit through MO2
  • From the top drop-down menu, select zMerge, and from the bottom select Fallout NV
  • Click Start Session
  • Click the Create Merge button
  • In the Details tab, change the Name to Content - Merge and make sure the Merge Method is Clean - Merge down (improved)
  • In the Plugins tab, select the following:
    • Character Expansions Revised - UW.esp
      Cut Sewers Restored.esp
      TMCOTT - Restored.esp
      Functional Post Game Ending - Uncut Wasteland Patch.esp
      Functional Post Game Ending - YUP Patch.esp
      Lucky38Overhaul.esp
      REPCONN Blues Delayer.esp
      YUP - AutumnLeaves Patch.esp
      Delay Autumn Leaves.esp
      TraitMod.esp
      Improved Traits.esp
      Baddarkl Perk Overhaul.esp
      Burden to Bear Armor.esp
      Content - Conflict Resolution AIO (All Modules).esp AND/OR whatever patches you selected from the Conflict Resolution FOMOD
  • Click OK to exit the merge menu
  • Click the Build button to the right of the merge name
  • The merge should build successfully and you can exit out of the program
  • Right-click in the left pane of MO2 and select All Mods -> Refresh
  • The merge should appear in the left pane and activate it like normal
  • Move Content - Merge.esp above Interior Lighting Overhaul - Ultimate Edition.esp in the right pane of MO2
  • Click the on the top bar of MO2 and select Merge Plugins Hide
  • Right-click on the Content - Merge listing, select Disable plugins, and then close
    • If the application fails to find your instance or points to another game, just click the folder icon and manually set them

Testing the Game

  • Run the game through the New Vegas option in MO2
  • Start up a new game and quickly run through the tutorial in Doc Mitchell's house
  • Run through the tutorial missions with Sunny Smiles to verify everything is running smoothly and correctly
