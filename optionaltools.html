These tools are mainly for advanced users looking to edit/create their own mods:

Paint.net

- Photo viewing/editing tool I always use to handle .dds files (the format used for texture files)

BSA Browser

- Allows you to browse files within .BSAs and extract them

FOMOD Creation Tool

- Easily create FOMOD installers for your mods

NifSkope

- Tool for basic creating and editing of meshes

Sniff

- Streamlined nif editor with preset operations

For Steam Users Only: GECK

- Tool that allows you to create/edit any data from the game. Comes preinstalled with the GOG version
