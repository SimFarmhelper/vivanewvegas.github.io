Install all of the following programs to the Tools folder in your Fallout New Vegas Modding Folder unless instructed otherwise

Microsoft VC++ 2013

Special install instructions:

  • Run the installer and follow the given instructions
    • If you get a message that the program is already installed, just exit out of the installer

- Required libraries for FNV and the 4GB Patcher

Microsoft VC++ 2015, 2017, 2019

Special install instructions:

  • Click the x86: vc_redist.x86.exe and x64: vc_redist.x64.exe links under the Visual Studio 2015, 2017 and 2019 header
  • Run both of the .exe files and follow the instructions to install them
    • If you get a message that the program is already installed, just exit out of the installer
    • If you are not on Windows 10, you may need to follow these instructions as well

- Required libraries for the mod manager for this guide, Mod Organizer 2 and NVGE

Mod Organizer 2

Special install instructions:

  • Download the Mod Organizer 2 (Archive) main file
  • Right-click on the downloaded archive and select Properties
  • In the General tab, check the box adjacent to Unblock
  • Select Apply then OK
    • If the option does not show up for you, ignore that step
  • Once you have extracted the archive, right-click on ModOrganizer.exe and select Properties
  • Navigate to the Compatibility tab and check the box adjacent to Run this program as an administrator
  • Select Apply then OK

- By far the best mod manager for all Bethesda games, takes advantage of a Virtual File System to keep your Data folder completely clean
  Join the Mod Organizer 2 team's Discord for developer builds (please don't go to their Discord and ask for troubleshooting with my guide! That's what my Discord is for)

A Note About Developer Versions

Links to developer versions of the tools listed in this section will be provided, but the instructions in the guide are specifically written for the public release verisons. I suggest only using the developer versions if you are familiar with the tools and can work out any discrepancies with the instructions in the guide.

MO2 Plugins

Special install instructions:

  • Extract the following from the archive to Mod Organizer 2 Path/plugins
    • /data
    • pySyncModOrder.py
    • pyMergePluginsHide.py

- Adds two new features to MO2, the ability to hide plugins that have been merged and the ability to sync profile mod orders with one another

Properly Updating Mod Organizer 2

For when a new version of Mod Organizer 2 is released:

  • Follow the same instructions above for unblocking the new archive
  • Extract the new archive into the MO2 folder and overwrite when prompted
  • Follow the same instructions above for running the program as an administrator
  • Re-install MO2 Plugins from above

FNVEdit

Install normally

- Advanced file viewer/editor and conflict detector. More commonly referred to as 'xEdit'
  Join the xEdit team's Discord for developer builds
  For a good tutorial (shouldn't be necessary if you are just following this guide, but if you want to learn more for yourself), see here

zMerge

Install normally

- Frontend tool similar to xEdit, will only be used in the guide to merge plugins. Will be referred to mostly as zMerge
  Check the linked Github page for new releases

FNVLODGen

Install normally

- One-click LOD generator
  Developer builds can be found on the same Discord as xEdit

BSA Decompressor

Special install instructions:

  • Do not install if you have your game on a slow HDD and/or are using/plan to use TTW (TTW is not a part of the guide, nor is it recommended for use with this guide)
  • Extract the downloaded archive to your modding folder
  • Right-click FNV BSA Decompressor.exe and select Run as administrator
  • Read and follow the instructions of the installer carefully, and exit the program when it's finished
  • Once you have exited, navigate to the Root folder and delete libogg.dll

- Decompresses BSAs and repacks them without zlib compression and fixes certain audio files that would not play in vanilla
NOTE: Verifying your game files through either Steam or GOG will undo the Decompressor's effects, so you must rerun it after

IMPORTANT NOTE:

If at any point during gameplay you encounter a lot of missing mesh errors (big red triangles), do the following:

  • Click the button on the toolbar of MO2 and select INI Editor
  • Navigate to the Fallout.ini tab
  • Find the line that starts with SArch=
  • Replace the entire line with the following (should all be on one line):
    • SArchiveList=Fallout - Invalidation.bsa, Fallout - Textures.bsa, Fallout - Textures2.bsa, Fallout - Meshes.bsa, Fallout - Meshes2.bsa, Fallout - Voices1.bsa, Fallout - Sound.bsa, Fallout - Misc.bsa

Ogg Vorbis Libraries

Special install instructions:

  • Extract the .dll files from the archive to the root folder and overwrite when prompted

- Latest compiled Ogg Vorbis Libraries that allow the game to play certain .ogg files it wouldn't be able to play otherwise

RivaTuner Statistics Server

Special install instructions

  • The download should start automatically after a few seconds
  • Run the installer and install the program wherever
  • Launch RTSS.exe
    • If no program appears, you may need to open it from the system tray
  • Once the program has launched, select Add in the bottom right
  • From the pop-up, navigate to the Root folder and select FalloutNV.exe
  • Select the newly created profile on the left-hand side of the window
  • Switch Start with Windows to On
  • Switch Show On-Screen Display to Off
  • Make sure Application detection level is Low
  • Set Framerate limit to 60
    • This guide will have New Vegas run in borderless windowed mode, so you must make sure G-Sync/Freesync is set up to work on windowed applications if you plan on using that
  • Switch On-Screen Display Support to Off
  • Minimize the program
    • Minimizing the program will minimize it to the tray, because it needs to be running in the background to work in-game. That means closing the program outright will make it not work. The program is set to automatically start minimized when your PC is started, so you don't need to worry about manually launching it every time you play
