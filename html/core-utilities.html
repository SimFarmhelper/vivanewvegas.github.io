A Note About Testing the Game

After each section in every module, you should launch the game and make sure everything is working correctly by running around for a few minutes

NVSE

Special install instructions:

  • Click the links next to the Download (stable version 5.1b4): and Download (unsupported releasefast 5.1b4): options
  • Extract all files ending in .dll and the .exe (5 total files) from the (stable version 5.1b4) archive to the Root folder
  • Extract the .dll file from the unsupported releasefast 5.1b4 archive to the Root folder and overwrite when prompted

- Extends the scripting capabilities of the engine and is essential for most mods

NVSE_Config.ini

Special install instructions:

  • Extract the nvse folder from the archive to the Data folder

- Increases heap allocation to help prevent stuttering/crashing

JIP LN NVSE Plugin

Special install instructions:

  • Download the mod manually
  • Extract the nvse folder from the archive to the Data folder

- NVSE extension with 800+ new functions

JohnnyGuitar NVSE

Special install instructions:

  • Download the mod manually
  • Extract the nvse folder from the archive to the Data folder

- Another NVSE extender

4GB Patcher

Special install instructions:

  • Extract the .exe from the archive to the root folder
  • Right-click on FalloutNVpatch.exe and select Run as administrator
  • When the command prompt appears, hit enter
  • The command prompt should now exit and FalloutNV_backup.exe should appear in the root folder
  • From now on, run the game through the New Vegas option in MO2
    • DO NOT run the game through the Fallout Launcher option, this will reset the Fallout.ini and FalloutPrefs.ini in MO2

- Makes the game large address aware, meaning it can use 4GB of RAM instead of 2GB
NOTE: Verifying your game files through either Steam or GOG will undo the patch, so you must rerun it after
NOTE 2: THIS PATCH IS STILL NEEDED WITH THE GOG VERSION OF THE GAME FOR THE NVSE INTEGRATION

Verifying NVSE and the 4GB Patch are Working

  • Run the game through MO2
  • Once the game has reached the title screen, open the console by pressing the Tilde (~) key
  • Input getnvseversion into the console and hit enter
    • The console should report NVSE version: 5
      • If it does not, verify NVSE is installed correctly and retry
  • Input getislaa into the console and hit enter
    • The console should report GetisLAA >> 2.0000
      • If the console reports GetisLAA >> 0.0000, the patch is not installed correctly
      • If the console reports GetisLAA >> 1.0000, the patch is installed correctly but is not working correctly, so you must run the game as an administrator
      • If the console reports an error message, you did not install JIP LN NVSE correctly

New Vegas Anti-Crash

Special install intructions:

  • Download the mod manually
  • Extract the nvse folder from the archive to the Data folder

- Implements structured exception handling and sanity checking to reduce the amount of crashes

New Vegas Tick Fix

Special install intructions:

  • Download the mod manually
  • Extract the nvse folder from the archive to the Data folder

Manual tweaking required:

  • Navigate to Data/nvse/plugins and open NVTF.ini
  • Set your iMaxFPS= (line 20) to your average FPS in-game + 60
    • For example, mine is set to 160 because I average around 100 FPS in-game
    • iMaxFPS is NOT an FPS cap, but rather a failsafe that disables the FPS fix if your FPS goes above what you have set for iMaxFPS. Therefore, it should be set to an FPS you won't get in-game much if ever

- Works to fix micro-stuttering and allows for higher framerates

OneTweak

Special install intructions:

  • Download the mod manually
  • Extract SKSE/plugins/OneTweak.dll and OneTweak.ini folder from the archive to Data/nvse/plugins

Manual Tweaking Required:

  • Click the button on the toolbar of MO2 and select INI Editor
  • Navigate to the FalloutCustom.ini tab
  • Add the following under the [Display] section
    • bFull Screen=0

- Adds fake fullscreen and dynamic process priority
