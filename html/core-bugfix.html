A Note About Load Order and Conflict Resolution

LOOT is not used in this guide. The best way to sort your load order is to use FNVEdit to view the conflicts and manually sort your load order based off that and create a custom conflict resolution patch. The guide will provide a fully-modular handmade conflict resolution FOMOD along with the proper load order for each module. This means that any mods listed can be left out. However, if you plan on adding any big mods, familiarize yourself with manually solving conflicts in xEdit. I suggest this tutorial for familiarizing yourself with the tool and this for making custom conflict resolution.

A Note About the Plugin Limit

Fallout New Vegas has a safe plugin cap of about ~110 (no one knows the exact number) compared to the cap of 255 in Skyrim/Fallout 4. The cap will also vary based on the size of loaded plugins. The guide will use the tool zMerge (a.k.a zEdit) to merge plugins together to stay below the limit. If you plan on adding your own mods in addition to the ones in the guide, familiarize yourself with the tool so you can stay below the limit as well.

Weapon Mesh Improvement Mod

Install normally


Manual FNVEdit editing required:

  • Run FNVEdit through MO2
  • Right-click in the Module Selection pop-up and click Select None
  • Check the box adjacent to WMIMNV.esp then hold down left shift and click OK
  • Wait for the message Background Loader: finished to appear in the right pane
  • Input 05000906 into the FormID search box in the top left and hit enter
    • If xEdit doesn't find anything, your load order is probably incorrect. Make sure the order of the official .ESMs are as follows:
        • FalloutNV.esm
        DeadMoney.esm
        HonestHearts.esm
        OldWorldBlues.esm
        LonesomeRoad.esm
        TribalPack.esm
        MercenaryPack.esm
        ClassicPack.esm
        CaravanPack.esm
        GunRunnersArsenal.esm
  • Right-click in the right pane and select Hide no conflict and empty rows
  • Drag and drop the value next to Tracer Chance from GunRunnersArsenal.esm on to the value from WMIMNV.esp
    • Fixes a bug that caused Dragon's Breath rounds to cause micro stutter when fired
  • Input 3a15e into the FormID search box in the top left and hit enter
  • Drag and drop WeapNVLightMachineGun "Light Machine Gun" [WEAP:000906DF] from FalloutNV.esm on to the blank box directly next to it from WMIMNV.esp
    • Adds LMGs back to the FormID list of weapons that Super Mutants can use to fix them spawning with them but not being able to use them
  • Input 121154 into the FormID search box in the top left and hit enter
  • Right-click on WMIMNV.esp on the top bar of the right pane and select Remove
    • ×
    • Removes absurd buffs to the Recharger Rifle
  • Save (control+s) and exit

- Fixes mesh errors, UV errors, incorrect flags, missing extra data, form lists, projectiles, and other weapon related bugs
NOTE: Due to the nature of this mod, any other mods that edit the same weapon meshes will be incompatible. This is why you won't see the likes of mods such as WMX or EVE
