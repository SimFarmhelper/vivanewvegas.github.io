A Note About Load Order and Conflict Resolution

LOOT is not used in this guide. The best way to sort your load order is to use FNVEdit to view the conflicts and manually sort your load order based off that and create a custom conflict resolution patch. The guide will provide a fully-modular handmade conflict resolution FOMOD for each module. This means that any mods listed can be left out, but if you plan on adding any big mods, familiarize yourself with manually solving conflicts in xEdit. I suggest this tutorial for familiarizing yourself with the tool and this for making custom conflict resolution. For more information about load orders and conflict resolution, see the FIXING CONFLICTS section here.

A Note About the Plugin Limit

Fallout New Vegas has a safe plugin cap of about ~110 (no one knows the exact number) compared to the cap of 255 in Skyrim/Fallout 4. The cap will also vary based on the size of loaded plugins. The guide will use the tool zMerge (a.k.a zEdit) to merge plugins together to stay below the limit. If you plan on adding your own mods in addition to the ones in the guide, familiarize yourself with the tool so you can stay below the limit as well.
