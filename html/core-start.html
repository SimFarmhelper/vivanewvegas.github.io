Requirements:

  • A copy of Fallout New Vegas with all the DLC from either Steam or GOG (There will be instructions for properly installing it in the next step)
  • Windows 7 or newer (64bit)
  • A text editor (I recommend Notepad++, but default Windows notepad will work)
  • An archiving tool (I recommend 7-Zip)
  • Latest GPU drivers (Either NVIDIA or AMD)
  • 15 gigabytes of free drive space (for just the Core module)
  • NOT fully compatible with TTW/New California out of the box. If you want to use this as a base for either of those mods, you must be aware of the incompatibilities (TTW and New California)
    • NOTE: TTW is not recommended for use with this guide, but if you decide to use it, come to me for support instead of bothering the TTW team
