  • Input QOL Improvements into the pop-up and select OK

lStewieAl's Tweaks

Files to install normally:

  • Main File: Stewie Tweaks
  • Optional File: Stewie Tweaks INI

Mod activation instructions:

  • Run the game and exit at the main menu
  • Double-click on Stewie Tweaks INI in the left pane of MO2
  • In the INI-Files tab, click on nvse/plugins/nvse_stewie_tweaks.ini
  • All the features of the mod can be turned on/off and tweaked here (they all come off by default) OR download my preset INI here
Tweak Effect
bBetterAutoWalk=1 Moving left/right no longer cancels autowalk
bCompanionPipColorChange=1 Shows companions on the compass as white instead of the HUD color
bConsoleNumpadSupport=1 Allows the use of the number pad in console
bDescriptiveMarkerAddedMessage=1 Says the location of added map markers in the pop-up message
bDontShowContainerAfterLockpick=1 Stops the container menu from opening automatically after picking a lock (mainly for use with the Loot Menu mod)
bFasterHackingTransition=1 Decreases the delay after successfully hacking a terminal from 3 seconds to 0.5 seconds
bFasterSaveMenuClose=1 Removes the 3 second delay from the pause menu after making a save
bGodmodeNeedsNoAmmo=1 Allows firing with 0 ammo while in god mode (for easy testing)
bHideInvisibleUndiscoveredLocations=1 Only show unvisited locations on the compass if they are marked on the map
bIgnoreCompanionsVATS=1 VATS no longer targets hired companions
bKeepRouletteBetAmount=1 Roulette bets no longer reset to 0 after every spin
bLockpickAllowMouse=1 Allows you to rotate locks by holding left mouse button
bMapShowUnconsciousCompanions=1 Shows unconscious companions on the local/world maps in red
bNoMinCompanionMapDistance=1 Removes the minimum distance for showing a companion on the world map
bNoSkillBooksAbove100=1 Prevents usage of skill books if the skill is already at 100 skill
bNoStartMenuDLCPopup=1 Removes the "Loading extra content" pop-up at the start menu
bNoVATSTargetInvisible=1 Prevents targeting of invisible enemies in VATS
bRapidReloadFix=1 Fixes reload speed bugs with the Rapid Reload and Hand Loader perks
bRemoveWeaponDamageBuffer=1 Removes the weapon damage buffer for weapons above 75% condition, like the game tutorial says
bTabClosesTerminals=1 Allows you to use the Tab key to close terminals
bUseWASDAsArrowKeys=1 Allows the use of WASD in place of arrow keys and E in place of the space bar in menus
bLevelUpScrollWheelSupport=1 Allows the use of the scroll wheel when allocating skill points in the level up menu
bTalkWhileSneakingIfShiftIsHeld=1 Makes holding shift while croucing talk to an npc instead of pick-pocketing
bAddInventoryDropItemHotkey=1 Allows you to also use Q to drop items in the Pip-Boy
bInstantContinueButton=1 Disables the "Continue from your last saved game?" prompt when clicking continue game
bDecreasedDialogueClickDelay=1 Shortens the delay before allowing clicking in the dialogue menu

bPickLocksEvenWithKey=1 Holding shift allows picking a lock even if you have the key
bConsoleHistoryNoStoreDuplicates=1 Console won't save duplicate console commands in the history
bSkipIntroVideo=1 Automatically skips the intro splash screens
bNumberedDialogHotkeys=1 Allows the use of the number keys to select the corresponding dialogue options
bNPCsDropWeaponHolsteredWeapon=1 Allows NPCs to drop weapons on death even if they are holstered
bCustomGameSettings=1 Enables the ability to change Gamesettings in the Stewie's Tweaks INI
bFasterTitleMenu=1 Speeds up the transition from splash screens to the main menu
bListMissingModsOnLoad=1 Lists missing plugins on the "Content is Missing" pop-up when loading a save
bPopupMenusDontMoveCursor=1 Stops pop-up menus moving the mouse to the center of the screen
bFixItemModMenuUnequip=1 Fixes a bug where closing the Pip-Boy while in the the Item Mod Menu would unequip the weapon if it was equipped
bUnequipBrokenArmor=1 Makes armor auto-unequip when broken
bNoKarmaMessages=1 Disables the good/bad karma messages and sounds
bNoCrippleCriticalMessages=1 Stops the cripple/critical corner messages from appearing
bNoAudioDistortion=1 Removes the audio distortion when NPCs are wearing masks
bFasterSleepWait=1 Decreases the delay between hours while waiting or sleeping
bHardcoreTweaks=1 Enables toggling of ammo weight, sleep healing and hired companions being essential
bNoQuestFailedPopup=1 Removes the pop-up and sound that appears when you fail a quest
bRememberConsoleHistory=1 Makes the console history persistent between play sessions
bDisableCompanionKillcam=1 Stops kill cams from activating on companion kills
bDisableInteriorFog=1 Disables non-static interior fog
bImprovedHacking=1 Prevents clicking on non-word characters from diminishing allowance
bScopeVisibleAPHP=1 Makes the AP and HP bars visible while scoped
bExtraConsoleDetails=1 Shows extra info about selected targets in the console
bHideMiscQuestItems=1 Hides quest items from the Misc tab in the Pip-Boy
bPickpocketOverhaul=1 Overhauls the pickpocket formula to be more logical and shows the success & chance
bShowInventorySortButton=1
bClickingActiveTabTogglesSorting=1
iMode = 1 		Clicking the active tab again in the Pip-Boy will sort the items by weight
bConsoleBackground=1 Adds a background to the console
bAddRGBSliders=1 Adds RGB sliders in the Display settings to control HUD color
bShowCurrencyInContainers=1 Shows currency (NCR/Legion) in containers
bEssentialCompanions=1 Hired companions are no longer able to die
bQuestFailedOnlyIfStarted=1 "Quest failed" pop-ups will still show up, but only if the failed quest was active
fAutoAimScreenPercentage=0 Removes auto aim
