Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input Core - Finish Line into the pop-up and select OK

Core - Conflict Resolution

Install normally


FOMOD Install Instructions:

  • Pick the All in One version if you installed all the mods listed in the FOMOD. If not, choose the Individual Patches option and choose all that apply to you

- Resolves conflicts between the mods from this module

Core - Load Order.txt

Special install instructions:

  • Extract loadorder.txt to Mod Organizer 2/profiles/Viva New Vegas - Core
  • Once the file is extracted, right-click in the left pane of MO2 and select All Mods -> Refresh

- Automatically ensures your load order is correct for the Core module
NOTE: This load order assumes you followed the entire guide with all the recommended options. For example, if you installed your own mods, those plugin-ins will be moved to the bottom of the LO and need to be manually moved back into the correct spot after the new load order is applied. It will still work perfectly even if you left out some mods too

Using zMerge

  • Verify Core - Conflict Resolution.esp or the individual patches you selected are in the lowest spot in your load order
  • Run zMerge through MO2
  • From the top drop-down menu, select zMerge, and from the bottom select Fallout NV
  • Click Start Session
    • Make sure you selected zMerge and not zEdit from the drop-down
  • Once zMerge is loaded, click the gear icon in the top right
  • In the Core tab, you can change the Application Theme to Night
  • In the Integration Settings tab, choose Mod Organizer 2 for Mod manager and Portable for the MO2 instance:
  • Hit the Detect button to automatically set your MO2 paths
    • If the application fails to find your instance or points to another game, just click the folder icon and manually set them
  • In the Merge Settings tab, make sure the Merge output path is pointed to your MO2 mods folder and both Disable plugins and Disable mods are un-checked
  • Click OK to exit the settings menu
  • Click the Create Merge button
  • In the Details tab, change the Name to Core - Merge and make sure the Merge Method is Clean - Merge down (improved)
  • In the Plugins tab, select the following:
    • ThrowableFixes.esp
      FIXforLegionArmors.esp
      Occulsion.esp
      JIP Improved Recipe Menu.esp
      Tutorial Killer.esp
      Simple DLC Delay.esp
      Inventory Search.esp
      Vanilla UI Plus.esp (Only if you followed the VUI+ HUD Section)
      Any mods you selected from the optional sections
      Core - Conflict Resolution AIO.esp OR whatever patches you selected from the Conflict Resolution FOMOD
  • Click OK to exit the merge menu
  • Click the Build button to the right of the merge name
  • The merge should build successfully and you can exit out of the program
  • Right-click in the left pane of MO2 and select All Mods -> Refresh
  • The merge should appear in the left pane and activate it like normal
  • Click the on the top bar of MO2 and select Merge Plugins Hide
  • Right-click on the Core - Merge listing, select Disable plugins, and then close
    • All the mods that were merged should now have their plugins disappear from the right pane

Rebuilding Merges with zMerge

For when a mod is updated/added/removed from the merge:

  • Click the on the top bar of MO2 and select Merge Plugins Hide
  • Right-click on the Core - Merge listing, select Enable plugins, and then close
  • Re-apply the loadorder.txt file from before
  • Run zMerge through MO2
  • From the top drop-down menu, select zMerge, and from the bottom select Fallout NV
  • Click Start Session
  • If you need to add/remove a mod from the merge, click Edit to the right of the merge name and select/deselect the desired plugins
  • Click the Build button to the right of the merge name
  • Exit zMerge, refresh MO2, and re-hide the plugins

Testing the Game

  • Run the game through the New Vegas option in MO2
    • If your game crashes when you launch it, make sure CardMaster NV.esp is hidden if you installed Caravan Card Control from Core - Optional HUD/UI
  • Start up a new game and quickly run through the tutorial in Doc Mitchell's house
  • Run through the tutorial missions with Sunny Smiles to verify everything is running smoothly and correctly
