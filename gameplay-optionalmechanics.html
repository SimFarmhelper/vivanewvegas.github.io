These are small and simple mods that can easily be added on top of any mod setup that don't require any conflict resolution and can be merged

Bypass Minigames

Install normally

- Allows you to bypass the lockpick and hack minigames if you have the proper skill by pressing the 'Z' key (still consumes a bobby pin)

Well Rested Overhaul

Install normally

- Improves upon the rather lackluster vanilla Well Rested perk

Better NCR Supply Cache

Install normally

- Fleshes out the loot pool of the NCR supply caches obtained through the NCR Radio to make them worthwhile

Honest Hearts Workbench Crate Luck

Install normally

- Randomizes the amount of skill books you receive from a certain container in HH based on your luck skill

Take Chems Make Fiends

Install normally

- Move freely among the Fiends with the Day Tripper perk if you wear Raider armor and a Fiend helmet

Recharger Rifle Rebalance with Scope Mod

Files to install normally:

  • Main File - Recharger Rifle Rebalance DLCs

- Re-works the Recharger Rifle to make it more viable and adds an attachable scope modification

Dead Money Toxic Cloud Fix

Install normally

- Changes the damage calculation formula for the toxic cloud in DM to take poison resistance into account
